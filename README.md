# Bijaya Adhikari — Portfolio

Full-Stack Developer · AI Engineer

Live at [bjadhs.github.io](https://bjadhs.github.io)

## What's Inside

- **Categorized skills section** — 6 cards with icons + proficiency meters
- **Dark & light mode** — toggle switch, preference persists
- **Project showcase** — image sliders, detail pages, live links
- **Contact form** — wired to backend API with validation
- **Responsive** — works on all screen sizes
- **Resume download** — served from backend

## Stack

Vanilla HTML, CSS, JavaScript. No frameworks, no build step.

## Run Locally

Open `index.html` in your browser.

```bash
git clone https://github.com/bjadhs/bjadhs.github.io.git
cd bjadhs.github.io
open index.html
```
